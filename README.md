

# Retail Sales Analysis

## About the Project

In this project, I analyzed a Retail Black Friday Sales dataset using Python.

The goal was to understand sales performance, customer behavior, product categories, and payment methods.

---

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## What I Did

### Data Understanding

* Explored the dataset
* Checked data types
* Reviewed basic statistics

### Data Cleaning

* Checked missing values
* Checked duplicate records
* Reviewed categorical columns

### Feature Engineering

Created a new column:

```python
discount_amount = original_price - final_price
```

Converted the purchase date column to DateTime format.

### Data Analysis

Analyzed:

* Sales by Product Category
* Sales by City
* Sales by Payment Method
* Sales by Customer Segment
* Sales by Month
* Black Friday Sales
* Weekend vs Weekday Sales

---

## Key Findings

* Electronics had the highest sales.
* Home & Kitchen ranked second.
* Mobile Wallet was the top payment method.
* Atlanta had the highest sales among cities.
* Returning customers generated the most revenue.
* November recorded the highest sales.
